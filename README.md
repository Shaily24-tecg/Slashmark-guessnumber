# Slashmark-GuessNumber

A simple Python-based number guessing game where the system generates a random number and the user has to guess it with hints.

## Features
- Random number generation
- User input based guessing
- Hint system (Too High / Too Low)
- Win condition on correct guess

## Requirements
- Python 3.x

## How to Run
```bash
python Guess#.py
```
